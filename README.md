# ai-builders

**誰でも・どんな業種でも、最高品質のAI体制を作って育てられる日本語スキル6点セット**

フリーランス・個人事業主・会社経営・会社員、業種を問わず使えます。AIの専門知識は不要 — 「何から始めればいいか分からない」段階から道案内し、各スキルが丁寧なヒアリングであなたの仕事とゴールを聞き切ってから、設計図を見せて、合意の上で構築します。

## 収録スキル（AI体制の一生をカバー）

| 工程 | スキル | 何ができるか | こう話しかける |
|---|---|---|---|
| 案内 | **ai-navigator** | 現在地とゴールを一緒に言語化し、成長マップで立ち位置を示して次の一歩を1つだけ案内 | 「何から始めたらいい？」「AIをどう使えばいいかわからない」 |
| 設計 | **ai-company-builder** | 事業全体のAI化を設計。ゴールの言語化→業務棚卸し→AI活用マップ→AI組織図→30/60/90日ロードマップ | 「AIで一人会社を回したい」「事業をAI化したい」 |
| 構築 | **agent-builder** | 自分専用のAIエージェント（毎日使う常設の相棒）を6要素で設計・構築 | 「自分専用のエージェントが欲しい」「秘書AIを作りたい」 |
| 構築 | **subagent-builder** | 作業を分担する部下エージェントを設計・生成。1体でも、複数体のチーム一括編成でも（体数の上限なし） | 「調査係の部下を作って」「部下を3人チームで作りたい」 |
| 構築 | **skill-builder** | 繰り返し作業（請求書・週報・発注チェック等）をあなた専用のスキルに変換 | 「この作業をスキルにしたい」「毎回頼むのが面倒」 |
| 運用・改善 | **ai-checkup** | 作ったAI体制の定期健康診断（周期は使い方に合わせて調整）。劣化・不調を診断書で示し、合意の上で治療 | 「AIの点検して」「最近調子が悪い」 |

6つは連携します：ai-navigatorが道案内→ai-company-builderが全体設計→agent-builder / subagent-builder / skill-builderが実装→ai-checkupが育てる。もちろんどれも単体で使えます。

## はじめての方へ

迷ったら、最初にこう話しかけてください：

> 「AIを仕事に使いたいんだけど、何から始めたらいい？」

ai-navigatorがあなたの現在地とゴールを一緒に整理して、次の一歩を案内します。

## インストール

### マーケットプレイス経由（推奨）

Claude Codeでこの2行を実行するだけです：

```
/plugin marketplace add shinichismile/ai-builders
/plugin install ai-builders@macchi-plugins
```

更新が出たら `/plugin marketplace update macchi-plugins` で最新版を受け取れます。

### ローカルで試す

```
claude --plugin-dir ./ai-builders
```

インストール後は普通に話しかけるだけで自動発動します。手動で呼ぶ場合は `/ai-builders:agent-builder` のようにプラグイン名を付けます。

## 設計思想

- **ゴールから始める**：AI化は手段。1年後どうなっていたら成功か（時間・お金・気持ち）を最初に言葉にしてから設計します
- **聞き切ってから作る**：ヒアリング項目を全部確認し、サマリー表で認識合わせしてから構築に入ります
- **一歩は小さく、レベルは飛ばさない**：成長マップ（話し相手→型を持つ→相棒と部下→AI組織）に沿って、今週始められる一歩から
- **人が決めて、AIが動く**：最終判断・責任・対外関係は常に人間側に置く設計です
- **作って終わりにしない**：AI体制は生き物のように劣化します。定期健康診断（ai-checkup）まで含めて1つのセットです
- **最小権限・安全第一**：シークレットの直書き禁止、破壊的操作の確認、業種規制（GDPR/HIPAA等）への配慮を全スキルに組み込み済み
- **中身は100%テキスト**：このプラグインに実行コード・自動実行フック・外部接続設定は一切含まれていません。全文がこのリポジトリで読める指示書のみで、動作は常にClaude Code本体の権限管理の下で行われます（企業のIT部門による監査も歓迎です）
- **公式仕様準拠**：サブエージェントのフロントマター仕様等はClaude Code公式ドキュメント（2026年7月時点）で裏取りしています
