# はじめてガイド — ゼロから使い始めるまで

Claude Codeを触ったことがない方向けの、いちばん最初の手順書です。すでにClaude Codeを使っている方は「3. プラグインを入れる」まで飛ばしてください。

## 1. Claude Codeを用意する

Claude Codeは、Anthropic社のAI「Claude」があなたのパソコンで実際に手を動かして働いてくれるツールです。ターミナル（黒い画面）版のほか、デスクトップアプリ版もあります。

- インストール方法は公式ドキュメントに従ってください（手順が更新されるため、常に公式が正です）：
  - 日本語: https://code.claude.com/docs/ja/overview
  - 英語: https://code.claude.com/docs/en/overview
- 利用にはAnthropicのアカウントとプラン（またはAPI利用）が必要です。料金は公式サイトでご確認ください

## 2. 起動する

インストール後、ターミナルで `claude` と打つ（またはデスクトップアプリを開く）とClaudeとの対話が始まります。最初に作業フォルダの確認などが出たら、画面の案内に従ってください。

## 3. プラグインを入れる（2行だけ）

Claude Codeの画面で、次の2行を1行ずつ入力します：

```
/plugin marketplace add shinichismile/ai-builders
```

```
/plugin install ai-builders@macchi-plugins
```

これで完了です。特別な設定は要りません。

## 4. 動作確認

入ったかどうか確かめたいときは：

```
/plugin
```

と打つと、インストール済みプラグインの一覧に `ai-builders` が表示されます。

## 5. 最初の一言

あとは普通に日本語で話しかけるだけです。何から始めるか迷ったら、まずこう言ってください：

> AIを仕事に使いたいんだけど、何から始めたらいい？

案内係（ai-navigator）が、あなたの現在地とゴールを一緒に整理して、次の一歩を教えてくれます。

もっと具体的な希望がある方は、[使い方ガイド（USAGE.md）](USAGE.md)の「こう話しかける」例を参考にどうぞ。

## 6. 更新を受け取る

このプラグインは改良が続きます。新しいバージョンが出たら：

```
/plugin marketplace update macchi-plugins
```

で最新版を受け取れます。

## 7. やめたくなったら

```
/plugin uninstall ai-builders@macchi-plugins
```

でいつでもきれいに外せます。プラグインを外しても、あなたがこのプラグインを使って**作ったもの**（エージェントやスキル）はあなたのものとして残ります。
