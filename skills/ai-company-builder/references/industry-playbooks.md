# 業種別プレイブック

該当業種の節を読み、ヒアリングとマップ作りの解像度を上げる。ユースケースは「よくあるクイックウィン→本命」の順。規制の注意は必ず設計に反映する（最新要件はWebSearchで裏取り）。

## 法律・士業（弁護士・行政書士・司法書士・社労士）

- クイックウィン: 判例・法令・官庁告示の調査要約 / 定型書類のドラフト / 顧客からの一次質問の回答案
- 本命: 契約書レビュー（リスク条項の洗い出し）/ 規制モニタリングの自動化
- 注意: 守秘義務。依頼者情報の外部送信範囲を最初に設計。**法的判断は必ず資格者本人**という関与点を全業務に。成果物には「検討材料であり法的助言ではない」の線引き

## 医療・介護・健康

- クイックウィン: 研究文献の整理・要約 / 研修資料・説明文書の作成 / 事務文書のドラフト
- 本命: カルテ・記録の要約補助 / コンプライアンス文書の管理
- 注意: 患者情報は最重要機密。HIPAA（米）・個人情報保護法の要配慮個人情報（日）等、地域の規制を必ず確認。**診断・治療判断にAIを単独で使わせない**

## 金融・会計・保険

- クイックウィン: 決算資料・市況レポートの要約 / 定型的な仕訳チェック / 顧客向け説明資料のドラフト
- 本命: 財務分析の一次処理 / リスク評価資料の作成 / コンプライアンス申告の下準備
- 注意: 金融規制（金商法等）・数字の正確性。**AIの計算・集計は必ず元データと突合する**関与点を置く。投資判断・与信判断は人間

## 教育・研修

- クイックウィン: 教材・テスト問題の生成 / 授業準備の資料集め / 保護者向け文書のドラフト
- 本命: 個別学習プランの設計補助 / フィードバックの自動下書き
- 注意: 生徒の個人情報・成績データの扱い。未成年データは特に慎重に

## EC・小売・マーケティング

- クイックウィン: 商品説明文・広告コピー・SNS投稿のドラフト / レビューの要約分析
- 本命: LP構成案の生成とA/Bテスト設計 / SEO記事の量産体制 / 売上データの週次分析
- 注意: 景品表示法（誇大広告）・ステマ規制。AI生成コンテンツも表示責任は事業者にある

## 製造・エンジニアリング

- クイックウィン: 仕様書・手順書・報告書のドラフト / 技術文書の整備
- 本命: コード生成・テスト自動化 / 図面・仕様の変更履歴管理の補助
- 注意: 技術情報・図面の機密性。取引先とのNDA範囲を確認してからデータを扱う

## クリエイティブ（デザイン・映像・執筆）

- クイックウィン: 企画のたたき台・リサーチ / ラフ案の言語化 / 請求・進行管理の事務
- 本命: デザイン批評の壁打ち役 / ブランドガイドラインの運用チェック / UXライティング
- 注意: 著作権。既存作家の模倣をAIにさせない。クライアントへのAI利用の開示方針を決めておく

## 店舗・サービス業（飲食・美容・小売店）

- クイックウィン: シフト表のたたき台 / SNS・販促物の作成 / よくある問い合わせの回答文
- 本命: 売上・客数データの週次分析と発注提案 / 予約・在庫のデータ整理
- 注意: 従業員・顧客の個人情報。レビュー返信をAIに書かせる場合も送信前に人が読む

## 不動産

- クイックウィン: 物件紹介文の作成 / 重要事項説明の下準備資料 / 問い合わせ一次対応のドラフト
- 本命: 市場調査・価格査定の一次分析 / 契約書類のチェック補助
- 注意: 宅建業法上の説明義務は資格者が果たす。査定はAI単独で確定させない

## 建設・工務店

- クイックウィン: 見積書・工程表のたたき台作成 / 現場写真の整理・報告書化 / 協力業者への連絡文テンプレ
- 本命: 施工計画・安全書類のドラフト整備 / 顧客向け進捗報告の定型化
- 注意: 建設業法（見積・契約書面の要件）。労働安全衛生関連の記録は正確性が最優先。個人情報（施主情報）の外部送信範囲を最初に設計
- 向くAI活用: 書類仕事の圧縮が本丸。現場作業そのものはAI化しない

## 一人社長・フリーランス共通（業種を問わず）

どの業種でも、一人でやっている場合は次の3部門から始めるのが定石：

1. **秘書部門**（常設エージェント）: メール下書き・スケジュール・タスク管理 → agent-builderへ
2. **調査部門**（サブエージェント）: 市場・競合・技術の調べ物 → subagent-builderへ
3. **経理補佐部門**（スキル＋定期実行）: 請求書作成・経費整理の定型処理 → skill-builderへ

この3つで「事務の時間」が目に見えて減ってから、業種固有の本命業務に進むと失敗しない。
